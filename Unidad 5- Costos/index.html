Unidad 5 — Costos

Resumen · Finanzas de Empresas 2025

¿Qué es un costo?

Valor sacrificado (en plata) para obtener bienes o servicios que se espera traigan beneficios presentes o futuros.

Si no genera beneficio → es un gasto, no un costo.

Costo del producto
Variable

Lo que cuesta producir ese bien (materiales, tiempo, MOD)

Se activa en el balance

Costo del período
Fijo

Costos del período independiente de la producción

Ej: alquiler, electricidad

Costos Totales = Costos del producto + Costos del período

Directos (CD)

Indirectos (CI)

Requiere asignación
Por comportamiento
Fijos (CF)

No varían con cambios en producción/ventas

A largo plazo se variabilizan

Varían a "saltos"

Variables (CV)

Varían con la producción/ventas

Corto plazo: lineales

Largo plazo: crecientes → decrecientes (economías de escala)

Semivariables

Parte fija + parte variable · Ej: sueldo fijo + comisión → se separan en CF y CV

Por función
Producción

MP + MOD + CIF

Distribución

Transporte, almacenamiento, publicidad

Administración

Personal admin, gastos de oficina

Financieros

Intereses, comisiones bancarias

Por decisión
Relevantes

Varían entre alternativas → influyen en la decisión

Irrelevantes (hundidos)

Iguales en todas las alternativas → no influyen

Métodos de asignación de costos

Rastreo directo — observación física o estequiométrica del producto
Rastreo por generadores — factor causal que mide el consumo del recurso (ej: horas/máquina)
Distribución — proporción arbitraria cuando no es posible rastrear
Tasa presupuestada CIF
Tasa CIF = CIF anual presupuestado / Cantidad anual del generador
CIF aplicado = Tasa CIF × Uso real del generador
Variación de indirectos
Subaplicación

CIF reales > CIF aplicados

Gastamos más de lo previsto

Sobreaplicación

CIF reales < CIF aplicados

Gastamos menos de lo previsto

Idea central del ABC

El producto no consume recursos directamente → consume actividades, y esas actividades demandan recursos y generan costos.

Las 3 etapas de asignación
1°
Asignación primaria

Distribuir CIF a departamentos productivos y de servicio

2°
Asignación secundaria

Redistribuir los depts de servicio en los productivos (empezando por el de mayor CIF)

3°
Asignación terciaria

Distribuir los CIF de depts productivos a cada producto según horas usadas en cada sector

¿Cuándo usar ABC?

No conviene

Un solo producto (todos los CI serían CD)

CIF muy relacionados con el volumen de producción

Los 3 tipos de costos en el sistema variable
CV de Comercialización (CVC)
Variable

Comisiones, transporte · Varían con el volumen de ventas

CVC = cvc × Q vendida
CV de Producción (CVP)
Variable

MOD, materiales directos · Varían con la producción

CVP = cvp × Q vendida
Costos Fijos (CF)
Fijo

Alquiler, salarios admin · Constantes independiente de la actividad

Fórmulas clave
CT = CF + cv × Q
Ventas = Precio × Q vendida
ROP = Ventas − CT = (P − cv) × Q − CF
mc = P − cv   (margen de contribución unitario)
MC = mc × Q vendida   (margen de contribución total)
ROP = MC − CF
Razones
Razón de recupero (rr)

Proporción del precio que cubre CV

rr = CV/V = cv/P
Razón de contribución (rc)

% del precio que queda como margen

rc = MC/V = mc/P
RO = rc × V − CF
Punto de nivelación

Cuando ROP = 0 → la empresa no gana ni pierde. Solo cubre CF y CV.

Zona pérdidas: V < CT · Zona ganancias: V > CT
Fórmulas de nivelación
Cantidad de nivelación
Q = CF / mc = CF / (P − cv)
Ventas de nivelación
V = CF / rc
Análisis de seguridad
Ventas / cantidad de seguridad

Máximo error posible en estimación de ventas

Vseg = Vest − Vniv
Qseg = Qest − Qniv
Margen de seguridad

Mayor margen = más tranquilidad

ms = Vseg / Vtotales
Empresa multiproductora

Múltiples combinaciones de productos dan el mismo MC

Solución: tratar la empresa como monoproductora usando promedio ponderado de cada producto

Q niv = CF / (Ppp − Cvpp)

Luego distribuir Qniv según el % de ventas de cada producto

Estado de resultados
Margen = Ventas − CMV = (P×Q) − (CU×Q)
RO = Margen − C.Adm − C.Comerciales − D&A

Costos sin consumir → Estado Patrimonial  ·  Costos consumidos → Estado de Resultados
